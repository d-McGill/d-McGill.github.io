Finally, I’ve had the time to focus on what I love doing: designing, coding and creating! I have re-designed my personal portfolio with other students and young professionals in mind. WordPress is a very popular and powerful platform. Using themes to create dynamic blogs and websites, I have created, designed and coded this theme from the ground up. I will soon be making this an open source project for other students and young professionals to contribute to and use the theme copywrite free. Every student should have some sort of online portfolio / blog online to show their talent, skill and journey. A GitHub repository will be created and stored on GitHub for people to contribute. Also, this version will be uploaded to wordpress.org for people to openly use for free! The project came about because I was looking to re-design my portfolio with a FREE theme (Because being a student is financially challenging) and I couldn’t find any suitable themes built for the interests of students and young professionals. If it doesn’t exist create it!
